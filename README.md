A professional creative portfolio website for **Richa Pitale**, Digital Marketing Manager & Creative Strategist based in Liverpool, UK.

[https://richapitale-sys.github.io/portfolio/](https://richapitale-sys.github.io/portfolio/)

## 📄 Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero section, featured projects teaser, about teaser, CTA |
| About | `about.html` | Professional profile, career timeline (2017–present), skills, languages, beyond work |
| Projects | `projects.html` | 4 case studies: Liverpool ONE, Liverpool Fed FC, Bombay Times Fashion Week, Mash Hospitality |
| Contact | `contact.html` | Contact information, email form with validation, social links |

---

## 🎨 Design Specs

- **Color Palette:** Dark charcoal `#1a1a1a` background with jewel-tone accents:
  - Emerald `#10b981`
  - Sapphire `#0ea5e9`
  - Deep Purple `#8b5cf6`
- **Typography:** Poppins (headings) + Inter (body) from Google Fonts
- **Layout:** Grid-based, card-based project display, centered content with max-width containers
- **Mobile:** Mobile-first responsive design with hamburger menu

## ✨ Features

- Semantic HTML5 with proper heading hierarchy
- Schema.org JSON-LD markup for Person and Portfolio
- Mobile-responsive hamburger navigation
- Smooth scroll animations (Intersection Observer)
- Contact form with client-side validation
- Scroll-to-top button
- SEO meta tags and canonical URLs
- Accessible: ARIA labels, roles, skip-friendly structure

---

## 🚀 Deployment (GitHub Pages)

1. Go to **Settings → Pages** in this repository
2. Set **Source** to `main` branch, root `/` folder
3. Click **Save** — the site will be live at `https://richapitale-sys.github.io/portfolio/`
